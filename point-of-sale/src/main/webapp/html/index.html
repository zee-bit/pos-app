Point-of-Sale

A POS or point of sale is a device that is used to process transactions by retail customers. The software for POS devices, provides features that allow retailers to monitor inventory and buying trends, create orders and track pricing accuracy.
This POS application is developed by Zeeshan Equbal, as internship project for Increff, under the mentorship of Akshit Mehta.
